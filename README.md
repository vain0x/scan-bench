# Scan Benchs

[![Build Status](https://travis-ci.org/vain0x/scan-bench.svg?branch=master)](https://travis-ci.org/vain0x/scan-bench)

## Input

```
a
b c
s
k
x1 x2 x3 ... xk
H W
#.#.#.#.#.#
.#.#.#.#.#.
#.#.#.#.#.#
N M
u1 v1 w1
u2 v2 w2
...
uM vM wM
```

### Constraints

- 1 ≤ a ≤ 100
- 1 ≤ b, c ≤ 1000
- 1 ≤ |s| ≤ 100

- 1 ≤ K ≤ 100
- -10^9 ≤ xi ≤ 10^9 (for 1 ≤ i ≤ K)

- 1 ≤ H, W ≤ 100

- 1 ≤ N ≤ 10^5
- 1 ≤ M ≤ 2 * 10^5
- 1 ≤ ui, vi ≤ N (for 1 ≤ i ≤ M)
- 1 ≤ wi ≤ 10^9
